# Форматы данных

У каждой модели есть два метода: predict и predict_verbose

## Метод predict

```python
predict(text: str)

return "text со всеми исправленными ошибками": str
```

### Пример return метода predict

```python
input: "а так хочеться что-то мочь менять в этом мире не обезательно менять но обязательно быть способным это зделать"
output: "А так хочется что-то мочь менять в этом мире: не обязательно менять, но обязательно быть способным это сделать"
```

## Метод predict_verbose

```python
predict_verbose(text: str)

return [
  [
    {
      "index": номер символа с которого началась ошибка: int,
      "error": слово с ошибкой: str,
      "suggestions": [
        список вариантов исправления ошибки в порядке уверенности: str 
      ],
      "message": тип ошибки: str
    },
  ],
  "text со всеми исправленными ошибками": str
]
```
